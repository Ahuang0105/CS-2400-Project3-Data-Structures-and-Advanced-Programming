The IDE used to build this programs is Eclipse, the zip file "AlanHuang_p3.zip" file
contain 1 folder call "CS2400 Project 3 int". Please place the whole folder under your 
"eclipse- workspace" folder. Next open your Eclipse software and selete "File" at the very
top left corner, next click "open Projects from file System". Under the "import source" 
please selet the path to "CS2400 Project 3 int" folder and clip "finish". Also, the source
code is locate in the folder call "bin" .

(1) Java interface and generic data type is not used in this project.

(2) BinarySearchTreeDemo.java is the Client program to test BinarySearchTree class for Task 1.
